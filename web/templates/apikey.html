How to get/use your BioPortal API Key ?


  • If you do not have a BioPortal account go to New Account  and create one.
  • Once you have the BioPortal account, login in BioPortal and go to your account details  and you should see your API Key there. The following image shows an screenshot of a BioPortal account with the API Key at the bottom.

  • You need to use your BioPortal API Key in your scripts. Example code that shows how to use the API Key with your SPARQL queries is avalaible here: Submitting SPARQL queries programmatically.
